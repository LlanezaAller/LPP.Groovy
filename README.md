# Instrucciones de uso

Parámetros para configurar la herramienta:
    En el fichero main.py:

        # 0 => MonteCarlo
        # 1 => BinaryTrees
        # 2 => Fibonacci
        mode = 0

        # 0 => Java
        # 1 => Groovy
        language = 1

        # Only for groovy
        # 0 => Dynamic
        # 1 => Static
        isStatic = 0

Para su ejecución utilizar python versión 3 o superior:

    python /main.py
